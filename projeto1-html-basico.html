Ferramentas utilizadas

A IDE Visual Studio Code é uma ferramenta open source e gratuita, disponível para Windows, Linux e Mac, também utilizada no para desenvolver páginas web. Embora também seja possível usar outros editores de texto simples, como Bloco de Notas, Sublime ou Notepad++.

Instalação

Para instalar o VSC, o processo é simples: Acesse o site Baixe o programa em visualstudio.com/download. Escolha a versão compatível com seu sistema operacional e clique em "Download". Após o download, abra o arquivo, aceite os termos de licença, defina a pasta de instalação e siga os passos até concluir.

No Windows, ao executar o programa pela primeira vez, uma mensagem de alerta pode aparecer devido ao uso de interfaces web pelo aplicativo. Basta clicar em "Permitir acesso" para prosseguir.

Interface

A tela inicial do VSC exibe uma mensagem de boas-vindas. A interface é composta por:

  • Explorador: Uma estrutura de pastas e arquivos no lado esquerdo.
  • Área de edição: O espaço central onde o código é escrito e editado.

Extensões

A extensão recomendada para instalar é o Live Server, que vai vai ajudar bastante no desenvolvimento. É uma extensão que atualiza automaticamente a página no navegador durante o desenvolvimento, facilitando os ajustes em tempo real. Outro plugin, que já vem pré-instalado dentro do VSC, é o Emmet, que já vem embutido dentro do VSC. O Emmet é um plugin que aumenta a produtividade ao permitir a criação rápida de código HTML e CSS com abreviações. Também é possível instalar no o Sublime ou outro programa que aceite plugins. O Emmet vai te ajudar na produtividade.

Inspetor de elementos

O inspetor de elementos é um recurso essencial para o desenvolvimento web disponível nos navegadores, como o Chrome. Abaixo estão os conceitos fundamentais:

  1. É uma ferramenta do desenvolvedor integrada aos navegadores (Chrome, Firefox, Safari, Edge, etc.) que permite inspecionar e modificar o HTML e o CSS de uma página web diretamente no navegador.
  2. Pode ser aberto pressionando a tecla F12 ou navegando pelo menu do Chrome (três pontinhos > Mais ferramentas > Ferramentas do desenvolvedor). Isso exibe um console com várias funcionalidades.
  3. Permite selecionar elementos na página (usando o ícone de seleção) para visualizar seu código HTML e os estilos CSS aplicados. Quando o mouse passa sobre um elemento, ele é destacado na página e seu código é exibido no painel.
  4. É possível alterar o HTML (como texto) e o CSS (como cores, tamanhos de fonte, etc.) em tempo real, mas essas mudanças são temporárias e revertidas ao atualizar a página. Isso é útil para estudo e experimentação.
  5. A ferramenta é recomendada para aprender HTML (estrutura da página) e CSS (formatação e estilo), ajudando a entender como os elementos são criados e estilizados.

Estrutura Básica do HTML

Um arquivo HTML é composto por três elementos principais:

  1. <html> Define o início e o fim do documento HTML.
  2. <head> Contém informações carregadas antes da exibição, como o <title>, que define o título exibido na aba do navegador.
  3. <body> Contém o conteúdo visível ao usuário, como textos e imagens.

Tags HTML são marcadas por < e >, e a maioria precisa de uma tag de fechamento com /.

Falando sobre Tags

O que são Tags HTML

Tags são elementos que definem a estrutura e o comportamento do conteúdo em uma página HTML.

Elas são delimitadas por < e >, e a maioria precisa de uma tag de fechamento () para indicar onde o efeito da tag termina.

Exemplo:

<i> para itálico e <strong> para negrito.

Tags com e sem Fechamento

Tags com fechamento:

São usadas para envolver conteúdo, como texto. Exemplo: <i>texto</i>.

Este texto, por exemplo está dentro de uma tag <h6>H6</h6>
Tags sem fechamento:

Também chamadas de tags auto-fechadas, não envolvem conteúdo e são usadas para inserir elementos únicos, como campos de entrada (<input>) ou imagens (<img>). Essas tags podem incluir uma barra / no final para indicar que são auto-fechadas.

Combinação de Tags:

Tags podem ser aninhadas para aplicar múltiplos efeitos. Por exemplo, usar <strong> e <i> juntos para criar texto em negrito e itálico.

A ordem de abertura e fechamento deve ser respeitada: a tag aberta por último deve ser fechada primeiro.

Atributos das Tags

Os atributos das tags são utilizados para modificar o comportamento e a aparência dos elementos em uma página web. São propriedades que uma tag HTML pode ter para definir características específicas.

Existem atributos genéricos (comuns a todas as tags) e atributos específicos (exclusivos de determinadas tags).

Atributos Genéricos:

  • id: Um identificador único para um elemento, usado principalmente para manipulação via JavaScript.
  • style: Permite aplicar estilos CSS diretamente no elemento, como alterar a cor ou o tamanho da fonte.
  • class: Usado para agrupar elementos que compartilham o mesmo estilo, facilitando a formatação via CSS.

Atributos Específicos:

  • type no input: Define o tipo de campo de entrada, como 'text' para texto, 'number' para números ou 'color' para um seletor de cor.
  • src no img: Especifica o caminho da imagem a ser exibida.
  • width no img: Define a largura da imagem em pixels.

Textos

As tags HTML específicas para formatação de texto são usadas para estruturar e estilizar conteúdo em uma página web. Seus principais conceitos são:

Tags de Título (Heading):

As tags <h1> a <h6> são usadas para definir títulos e subtítulos, onde <h1> é o mais importante e <h6> o menos. O navegador aplica automaticamente espaçamentos (margens) entre esses elementos para indicar hierarquia.

Parágrafos:

A tag <p> é usada para definir parágrafos de texto. Assim como as tags de título, o navegador adiciona margens automaticamente para separação visual.

Citações:

A tag <blockquote> é utilizada para destacar citações ou observações, geralmente indentadas para diferenciá-las do texto principal.

Formatação Inline:

Tags como <strong> (negrito), <u> (sublinhado), <i> (itálico), <mark> (marca-texto), <sup> (sobrescrito, ex.: x2 para expoentes) e <sub> (subscrito, ex.: H2O para fórmulas químicas) são usadas para aplicar estilos específicos a partes do texto.

Responsividade e Inspeção:

O navegador ajusta automaticamente o layout do texto para diferentes tamanhos de tela, mas o CSS é necessário para um design mais responsivo.

O inspetor de elementos do navegador permite visualizar e entender as margens e estilos aplicados automaticamente.

Recursos Adicionais:

O site W3Schools é recomendado como uma referência completa para aprender mais sobre todas as tags HTML disponíveis.

Listas Ordenadas e Não Ordenadas

As listas ordenadas e não ordenadas em HTML são estruturadas e suas aplicações práticas. Seus principais conceitos são:

Tipos de Listas:

Lista Ordenada (<ol>)

Utiliza a tag <ol> (ordered list) e exibe itens com numeração sequencial (1, 2, 3, etc.). É ideal para conteúdos que seguem uma ordem específica, como um sumário.

Lista Não Ordenada (<ul>)

Utiliza a tag <ul> (unordered list) e exibe itens com marcadores (geralmente bolinhas). É usada quando a ordem dos itens não é relevante.

Estrutura das Listas:

Ambas as listas requerem a tag <li> (list item) como "filha" para definir cada item da lista.

As tags <ol> e <ul> são tags "pai" que não aceitam texto diretamente; os itens devem ser colocados dentro de <li>.

Listas Aninhadas:

É possível criar listas dentro de outras listas (aninhamento), como uma <ol> dentro de um <li>, para criar subitens (ex.: 3.1, 3.2).

Links

Os principais fundamentos dos links em HTML, usando a tag <a> e seus atributos.

Função dos Links:

Links (ou âncoras) permitem navegar entre páginas, conectando uma página a outra, seja no mesmo site ou em sites externos. A tag <a> (anchor) é usada para criar links.

O atributo href (hypertext reference) especifica o destino do link, que pode ser uma URL externa (ex.: https://dio.me) ou um arquivo local (ex.: about.html).

Atributo target:

Define como o link será aberto:

  • _self: Abre na mesma aba (comportamento padrão).
  • _blank: Abre em uma nova aba.
Boas Práticas de Navegação:
  • Links para páginas do mesmo site devem abrir na mesma aba (_self) para manter o usuário no site.
  • Links para sites externos devem abrir em nova aba (_blank) para não interromper a navegação no site original.
  • Outros valores como _parent e _top são mencionados como obsoletos, usados em frames para retrocompatibilidade.

Atributo title:

Adiciona um texto descritivo que aparece como um balãozinho (tooltip) quando o mouse passa sobre o link, melhorando a experiência do usuário.

Recursos do Navegador:

Mesmo com target="_self", o usuário pode forçar a abertura em nova aba usando Ctrl + clique ou em nova janela com Shift + clique.
